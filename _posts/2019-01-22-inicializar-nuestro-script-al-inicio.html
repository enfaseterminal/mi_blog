--- layout: post title: Inicializar nuestro script al inicio del sistema date: '2019-01-22T19:36:00.000+01:00' author: David P. tags: - update-rc.d - linux - cp - chmod modified_time: '2019-02-11T19:37:46.907+01:00' blogger_id: tag:blogger.com,1999:blog-3624480011084378437.post-8545676074947152052 blogger_orig_url: https://www.enfaseterminal.com/2019/01/inicializar-nuestro-script-al-inicio.html --- Resulta interesante arrancar nuestro script cada vez que se inicie nuestro sistema operativo basado en Linux.
Daremos privilegios de ejecución
sudo chmod +x script.sh
Copiaremos nuestro  script, con el comando cp,  a la carpeta /etc/init.d/
sudo cp script.sh /etc/init.d/
Con el comando update-rc.d añadiremos el archivo en el arranque de Linux.
sudo update-rc.d script.sh defaults
